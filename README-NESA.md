# Minecraft Nesa Explorer 5 Edition

A 3D Minecraft-style game built with Three.js, featuring:

## Features
- **1024x1024 world** - Massive Beta terrain generation
- **Flying mode** - Creative flight with Space/Shift controls
- **Inventory system** - Minecraft Beta-style inventory with texture previews
- **Fast Mode** - Toggle 720p resolution, disable fog, 64 block render distance
- **Windows 98 aesthetic** - Retro frame and styling
- **No mobs, no gravity** - Pure creative building experience
- **27+ block types** - Including ores, wood, stone, glass, flowers, and more

## Controls
- **WASD** - Move
- **Mouse** - Look around
- **Space** - Fly up
- **Shift** - Fly down
- **I** - Open inventory
- **1-9** - Hotbar slots
- **Left Click** - Break blocks
- **Right Click** - Place blocks
- **ESC** - Settings menu
- **F5** - Toggle third-person view

## Deployment
This game is ready to deploy on Render as a static site.

## Local Development
Simply open `vanilla.html` in a web browser, or run:
```bash
python -m http.server 8080
```
Then visit http://localhost:8080/vanilla.html
